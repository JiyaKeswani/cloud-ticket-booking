# ☁️ SkyPass

### Cloud-Powered Event Ticketing Platform

SkyPass is a full-stack cloud-native event ticket booking platform that enables users to register, log in securely, browse events, book tickets, and view booking history.

## Features

* User Registration & Login
* JWT Authentication & Authorization
* Secure Password Hashing with bcrypt
* Event Management
* Ticket Booking System
* Booking History Tracking
* PostgreSQL Database (Neon Cloud)
* RESTful API Architecture

## Tech Stack

### Backend

* Node.js
* Express.js
* PostgreSQL
